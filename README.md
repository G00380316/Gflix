# Installation Guide

## Step 1: Download and install the repository zip
1. Open Kodi.
2. Go to **Settings**.
3. Open **Add-ons**.
4. Enable **Unknown sources** if prompted.
5. Go back and open the **Add-on Browser** (package icon).
6. Select **Install from zip file**.
7. Browse to the downloaded file:
   - `repository.G00380316-3.0.3.zip`
8. Wait for the repository installed notification.

Kodi’s documentation confirms that zip installation from private repositories requires **Unknown sources** to be enabled. :contentReference[oaicite:1]{index=1}

## Step 2: Install the add-on
1. From the **Add-on Browser**, choose **Install from repository**.
2. Open **G00380316 Repository**.
3. Open **Video add-ons**.
4. Select **GIPTV**.
5. Press **Install**.
6. Wait for the add-on installed notification.

# GIPTV Setup Guide

After installing GIPTV:

1. Open **GIPTV**.
2. Open the add-on **Settings**.
3. Choose the account slot you want to use:
   - Account 1
   - Account 2
   - Account 3
4. Enter your IPTV provider details:
   - **Server**
   - **Username**
   - **Password**

## Optional settings
You can also configure:

- **Buffer Size**
  - Small
  - Medium
  - Large
  - Very Large

- **Stream Format**
  - Auto
  - TS
  - M3U8

- **EPG URL**
  - Optional custom EPG source
  - You can also clear it with the provided setting action

- **EPG Offset**
  - Use this if guide times are early or late

- **Catch-up Offset**
  - Use this if catch-up playback starts too early or too late

## Cache
In **General Settings**, you can enable or disable cache behavior.

## Extra tools
In **More Settings**, you can:
- Clear cache
- Build search index
- Close settings

# First-Time Recommended Setup

For the best experience after adding your account:

1. Open **Settings**
2. Enter your provider credentials
3. Set your preferred **Buffer Size**
4. Choose your preferred **Stream Format**
5. Add an **EPG URL** if your provider requires one
6. Run **Build Search Index**
7. Reopen the add-on and start browsing

---

# Notes

- GIPTV works with provider credentials that support Xtream-style login details.
- Playback quality and speed depend on your provider, device, and connection.
- If guide times are off, adjust **EPG Offset**.
- If catch-up timings are off, adjust **Catch-up Offset**.
- If something appears stale or slow, try **Clear Cache** and then **Build Search Index**.

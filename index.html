G00380316 Kodi Repository

A Kodi repository for installing GIPTV and other included add-ons more easily.

Download Repository Zip

Download the repository zip file below, then install it in Kodi using Install from zip file.

Download repository.G00380316-2.9.3.zip

How to Install

  1. Open Kodi
  2. Go to Settings → Add-ons
  3. Enable Unknown sources if prompted
  4. Open the Add-on Browser (package icon)
  5. Select Install from zip file
  6. Choose the downloaded file:
    repository.G00380316-2.9.3.zip
  7. After the repository installs, go to Install from repository
  8. Open G00380316 Repository
  9. Open Video add-ons
  10. Select GIPTV and press Install

How to Configure GIPTV

  1. Open GIPTV
  2. Open Settings
  3. Select the account slot you want to use
  4. Enter:
    • Server
    • Username
    • Password
  5. Optionally configure:
    • Buffer Size
    • Stream Format
    • EPG URL
    • EPG Offset
    • Catch-up Offset
  6. Use Build Search Index after setup for faster searching

Gflix Repository

Official source for Gflix Addons.

Installation Guide

  1. In Kodi, go to **Settings** (gear icon) -> **File Manager**.
  2. Select **Add source**.
  3. Enter the URL for this repository: https://G00380316.github.io/Gflix/
  4. Name it **Gflix Repo**.
  5. Return to **Add-ons** > **Install from zip file** and select the **Gflix Repo** source.
